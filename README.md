# PiicoDev® Adapter for Breadboards

This is the hardware repo for the Core Electronics [PiicoDev Adapter for Breadboards](https://core-electronics.com.au/catalog/product/view/sku/CE07691).

The PiicoDev® Adapter for Breadboards allows you to break-out the 4-wires in a PiicoDev cable for prototyping on a breadboard.

**Features**

- Breaks out each PicoDev pin, with handy labels
- Breadboard compatible 2.54mm header
- PiicoDev connector, compatible with Qwiic and STEMMA QT (3.3V only)
- Manufactured and supported in Australia by Core Electronics

**Technical Details**

- Dimensions: 12.9x10.2mm
- PiicoDev connector pinout:
  - 1 GND
  - 2 3.3V
  - 3 SDA
  - 4 SCL
